# portal-index-lab

本仓库用于归档和发布多个独立的 HTML 页面。这些页面通常作为索引、导航或信息展示用途，不针对任何特定域名或具体网站。

## 项目简介

portal-index-lab 是一个轻量级的前端页面集合仓库。每一份 HTML 文件均可独立运行，无需依赖外部框架或后台服务。仓库的主要目标是提供一个结构化的存放空间，便于不同页面的版本管理、预览与共享。

## 目录说明

仓库根目录下按功能或主题划分文件夹，每个文件夹内包含一个或多个独立的 HTML 文件。典型结构如下：

```
/
├── page-group-a/
│   ├── index.html
│   └── style.css
├── page-group-b/
│   └── index.html
└── README.md
```

- 每个子文件夹代表一类页面集合，文件夹名称反映其内容主题。
- 页面内引用的资源（如 CSS、JS）尽量使用相对路径或内联方式，以保持独立性。

## 页面归档说明

- 所有页面均为静态 HTML，可直接在浏览器中打开运行。
- 页面内不包含任何外部跟踪、分析脚本或商业化内容。
- 归档的页面可能包含占位性文字、示例数据或通用样式，仅供演示或测试使用。
- 部分页面可能引用公开的第三方库（如字体、图标），这些库的版权归其原作者所有。

## 维护说明

- 欢迎提交 Pull Request 以添加新的独立页面或改进现有页面。
- 新增页面请遵循目录结构规范，并确保页面可独立运行。
- 不鼓励在页面中嵌入与归档目的无关的链接或内容。
- 仓库维护者会不定期清理不再适用的页面或调整目录结构。
